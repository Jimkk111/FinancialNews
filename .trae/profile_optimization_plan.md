# Profile.vue 页面优化计划

## 一、现状问题

Profile.vue 与项目整体风格存在以下差异：

| 问题       | 当前实现                                            | 项目规范                   |
| -------- | ----------------------------------------------- | ---------------------- |
| **头部背景** | `bg-gradient-to-br from-blue-600 to-purple-600` | 纯白背景 + 品牌红元素           |
| **头像区域** | 渐变背景 + 半透明遮罩                                    | 应简洁，与其他页面一致            |
| **图标颜色** | 多彩图标（blue/purple/red/orange）                    | 应统一为灰色系或品牌红            |
| **菜单卡片** | `rounded-2xl shadow-sm`                         | 扁平化风格，无圆角大阴影           |
| **分割线**  | `border-gray-100` + 复杂结构                        | 简洁的 `h-px bg-gray-200` |
| **整体风格** | 类似社交App（Instagram风）                             | 财经新闻风格（财联社风）           |

## 二、优化方案

### 1. 头部区域重设计

```
当前: 蓝色渐变背景 + 白色文字
优化: 白色背景 + 顶部红色细线装饰 + 灰色文字
```

**具体改动：**

- 移除渐变背景，改为 `bg-white`
- 用户名颜色改为 `text-gray-900`
- 添加顶部 `border-t-4 border-[#D92E2E]` 红色装饰条
- 头像点击区域保持圆形但去掉半透明背景

### 2. 菜单列表重设计

```
当前: 圆角卡片 + 彩色图标 + 阴影
优化: 扁平列表 + 灰色图标 + 细分割线
```

**具体改动：**

- 卡片改为纯列表结构，参考 Collection.vue 样式
- 图标颜色统一改为 `text-gray-500`
- 分割线改为 `h-px bg-gray-200`
- 移除 `rounded-2xl shadow-sm`
- 文字颜色统一：`text-gray-900`

### 3. 功能入口简化

当前只有「我的收藏」和「阅读历史」可用（其他功能 `hidden: true`），优化后：

- 只保留这两个可用功能，删除所有被隐藏（`hidden: true`）的菜单项代码
- 简化菜单结构，去除复杂的分组

### 4. BottomNav 调整

移除底部导航栏中的「收藏」和「历史」入口，保留「首页」「AI」「我的」三个导航项。

## 三、优化后的布局结构

```
┌─────────────────────────────────┐
│ ████ 红色顶部条 ████           │ ← border-t-4 border-[#D92E2E]
├─────────────────────────────────┤
│  [头像80px]  用户名             │ ← bg-white, text-gray-900
│              点击进入个人信息 → │
├─────────────────────────────────┤
│  我的服务                        │ ← text-sm font-semibold text-gray-500
│  ┌─────────────────────────────┐│
│  │ [心形] 我的收藏        [>]  ││ ← text-gray-500 + text-gray-900
│  ├─────────────────────────────┤│ ← h-px bg-gray-200
│  │ [书形] 阅读历史        [>]  ││
│  └─────────────────────────────┘│
├─────────────────────────────────┤
│  财经快讯 v1.0.0                │ ← text-gray-400 text-xs
├─────────────────────────────────┤
│  [首页] [AI] [我的]              │ ← BottomNav 固定底部（移除收藏和历史）
└─────────────────────────────────┘
```

## 四、涉及改动

| 文件                         | 改动内容                           |
| ---------------------------- | ---------------------------------- |
| `src/views/Profile.vue`      | 样式优化 + 删除隐藏菜单项代码       |
| `src/components/BottomNav.vue` | 移除「收藏」和「历史」导航入口     |

## 五、注意事项

1. **仅样式优化**：不改变组件的 props/emit 接口
2. **保持功能**：收藏、历史跳转逻辑不变
3. **响应式兼容**：保持移动端优先设计
