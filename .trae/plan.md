# 财联社风格首页布局实现方案

## 一、设计目标

基于财联社 App 的视觉风格，重新设计项目的首页布局，打造专业、紧凑、信息密度高的财经新闻阅读体验。

***

## 二、视觉风格定位

### 2.1 设计方向：**专业财经媒体风格**

参考财联社的设计特点：

- **高信息密度**：紧凑布局，最大化内容展示
- **清晰的视觉层次**：通过字号、字重、颜色区分重要性
- **克制的配色**：中性色为主，红色作为强调色（财联社品牌色）
- **功能优先**：装饰性元素最小化，专注于内容传达

### 2.2 配色方案

```css
/* 主色调 */
--primary-red: #D92E2E;        /* 财联社红，用于强调、热榜 */
--primary-dark: #1A1A1A;       /* 近乎黑的深灰，用于标题 */
--text-primary: #333333;       /* 主要文字 */
--text-secondary: #666666;     /* 次要文字（来源、时间） */
--text-muted: #999999;         /* 弱化文字（阅读量） */

/* 背景色 */
--bg-primary: #FFFFFF;         /* 主背景 */
--bg-secondary: #F5F5F5;       /* 分割线、卡片背景 */
--bg-tertiary: #FAFAFA;        /* 悬停背景 */

/* 功能色 */
--success-green: #00A859;      /* 上涨（A 股绿色） */
--danger-red: #D92E2E;         /* 下跌（A 股红色） */
--warning-orange: #FF8800;     /* 警告、广告标识 */
```

### 2.3 字体排印

```css
/* 字号体系（基于 8px 网格） */
--text-xs: 12px;               /* 辅助信息 */
--text-sm: 14px;               /* 正文、来源 */
--text-base: 16px;             /* 标题 */
--text-lg: 18px;               /* 大标题 */
--text-xl: 20px;               /* 分类导航 */

/* 字重 */
--font-normal: 400;            /* 正文 */
--font-medium: 500;            /* 来源、标签 */
--font-semibold: 600;          /* 标题 */
--font-bold: 700;              /* 分类导航、重要标题 */
```

## 三、布局结构设计

### 3.1 整体布局

采用**移动端优先**的单列布局：

```
┌─────────────────────────────────┐
│    Header（Logo + 头像）          │  ← 固定顶部
├─────────────────────────────────┤
│      SearchBar（搜索栏）          │  ← 独立组件
├─────────────────────────────────┤
│      CategoryTabs（分类导航）     │  ← 可横向滚动
├─────────────────────────────────┤
│                                 │
│        NewsList（新闻列表）       │  ← 无限滚动
│                                 │
│  ┌─────────────────────────┐    │
│  │ 新闻卡片 1               │    │
│  ├─────────────────────────┤    │
│  │ 新闻卡片 2               │    │
│  ├─────────────────────────┤    │
│  │ 新闻卡片 3               │    │
│  └─────────────────────────┘    │
│                                 │
├─────────────────────────────────┤
│      BottomNav（底部导航栏）      │  ← 固定底部
└─────────────────────────────────┘
```

### 3.2 Header 组件重构

**布局结构：**

```
┌──────────────────────────────────────────────┐
│  [左侧待定]    财经新闻 Logo    [用户头像]    │
└──────────────────────────────────────────────┘
```

**实现细节：**

- 高度：`56px`（符合移动端触摸区域标准）
- 布局：Flexbox 三段式布局（左侧 | 居中 Logo | 右侧头像）
- 背景：白色
- 边框：底部 `1px solid #F0F0F0`
- Logo：品牌红色 `#D92E2E`，字重 `700`
- 头像：圆形，`32px × 32px`，点击跳转个人中心

**待确认项：**

- 左侧区域：可考虑放置消息通知、扫一扫等功能图标

### 3.3 CategoryTabs 组件优化

**当前问题：**

- 样式风格不够突出，缺少财经媒体的紧凑感
- 激活态不够明显

**改进方案：**

```
  电报  头条  A 股  港股  环球  公司  基金  更多 +
  ──    ──    ──    ──    ──    ──    ──
        ↑
      红色下划线 + 加粗
```

**实现细节：**

- 默认字体大小：`15px`
- 激活态：字重 `700` + 底部红色下划线 `2px`
- 间距：左右 `12px`，确保可横向滚动
- 添加"更多"按钮展开额外分类

### 3.4 NewsList 新闻卡片重构（核心改动）

**财联社风格卡片布局：**

```
┌─────────────────────────────────────────┐
│                                         │
│  阿里将造"养虾"PC 与主机硬件              │  ← 标题（16px, 600）
│  （最多 2 行，超出省略）                   │
│                                         │
│  ┌─────────────┐                        │
│  │             │                        │
│  │   配图      │  蓝鲸新闻 1 小时前       │  ← 来源 + 时间（12px, 400）
│  │             │  5w+ 已阅读             │  ← 阅读量（12px, 400, 右对齐）
│  └─────────────┘                        │
│                                         │
├─────────────────────────────────────────┤
│  [广告] 集合竞价暗藏涨停密码... >>>      │  ← 广告标识（橙色标签）
└─────────────────────────────────────────┘
```

**用户要求：卡片高度固定**

**实现细节：**

1. **卡片容器**
   - 背景：白色
   - 内边距：`16px`
   - 分割线：`1px solid #F0F0F0`
   - 悬停效果：背景变为 `#FAFAFA`
   - **固定高度：`140px`**（确保视觉统一）
2. **标题样式**
   - 字号：`16px`
   - 字重：`600`
   - 行高：`1.5`
   - 最多显示 2 行，超出部分省略
   - 颜色：`#1A1A1A`
3. **配图规格**
   - 尺寸：`120px × 80px`
   - 圆角：`4px`
   - 位置：右侧（有图）/ 无图（标题占满宽度）
   - 懒加载：添加 `loading="lazy"`
4. **元信息布局**
   - 来源：`12px`，`#666666`
   - 时间：相对时间格式（"1 小时前"）
   - 阅读量：右对齐，`12px`，`#999999`
   - 间距：`8px` 垂直间距
5. **广告标识**
   - 标签：`[广告]` 橙色背景 `#FF8800`
   - 位置：卡片左下角
   - 字号：`10px`

### 3.5 BottomNav 底部导航栏

**当前设计已符合要求，仅需微调：**

```
  🏠      ▶️      👑      📊      📈
 首页    视频     VIP     盯盘    投研
  ↑
红色（激活态）
```

**调整：**

- 图标大小：`24px`
- 字号：`10px`
- 激活态：红色 `#D92E2E`
- 未激活：灰色 `#999999`

## 四、组件改造清单

### 4.1 需要修改的组件

| 组件                 | 改动幅度 | 主要改动         |
| ------------------ | ---- | ------------ |
| `Header.vue`       | 中    | 三段式布局、Logo 样式、头像点击 |
| `SearchBar.vue`    | 小    | 独立组件样式优化   |
| `CategoryTabs.vue` | 中    | 导航样式、激活态指示器  |
| `NewsList.vue`     | 大    | 卡片布局重构、固定高度、配图优化 |
| `BottomNav.vue`    | 小    | 图标间距、激活态颜色   |

### 4.2 新增功能

1. **图片懒加载**
   - 在 `NewsList.vue` 中为配图添加 `loading="lazy"`
   - 考虑添加占位图（低质量模糊效果）
2. **阅读量格式化**
   - 新增工具函数 `formatViews(count: number): string`
   - 规则：10000+ → "1w+", 1000+ → "1k+"
3. **相对时间显示**
   - 复用现有的 `formatTime` 工具函数
   - 确保输出格式符合财经媒体风格（"1 小时前"、"39 分钟前"）

## 五、技术实现细节

### 5.1 Tailwind CSS 配置

**需要扩展的样式工具类：**

```typescript
// tailwind.config.ts（如有需要）
theme: {
  extend: {
    colors: {
      'finance-red': '#D92E2E',
      'finance-dark': '#1A1A1A',
      'finance-gray': '#666666',
      'finance-muted': '#999999',
    },
    fontSize: {
      'xxs': ['10px', { lineHeight: '14px' }],
      'news-title': ['16px', { lineHeight: '24px', fontWeight: '600' }],
    },
  }
}
```

### 5.2 响应式考虑

虽然项目是移动端优先，但仍需考虑平板适配：

```css
/* 移动端（默认） */
.card { padding: 16px; }

/* 平板（≥768px） */
@media (min-width: 768px) {
  .card { padding: 20px; }
  .news-title { font-size: 18px; }
}
```

### 5.3 性能优化

1. **图片优化**
   - 使用 `loading="lazy"` 懒加载
   - 考虑使用 WebP 格式（如果 picsum 支持）
2. **列表渲染优化**
   - 保持现有的 IntersectionObserver 无限滚动
   - 考虑添加虚拟滚动（如果列表超过 50 条）
3. **CSS 优化**
   - 使用 Tailwind 的原子化 CSS，避免重复样式
   - 移除未使用的样式（生产构建自动处理）

## 八、风险与注意事项

### 8.1 已知风险

1. **图片加载失败**
   - 外部图片 URL（picsum.photos）可能不稳定
   - 解决方案：添加错误处理，显示默认占位图
2. **样式冲突**
   - 全局样式可能影响新组件样式
   - 解决方案：使用 scoped CSS 或 CSS Modules
3. **浏览器兼容性**
   - IntersectionObserver 在旧浏览器可能不支持
   - 解决方案：添加 polyfill 或降级方案

### 8.2 注意事项

1. **保持现有功能**
   - 确保改造不影响现有的收藏、历史功能
   - 新闻详情跳转逻辑保持不变
2. **数据层不变**
   - 仅修改 UI 层，不改动 `data/` 和 `services/` 目录
3. **TypeScript 类型完整**
   - 所有新增代码保持类型完整
   - 不使用 `any` 类型
